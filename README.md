## Beaverlib

**What is Beaverlib?** <br>
Beaverlib is a library built in Kotlin, designed to simplify the reuse of code from year to year, 
in addition to other helpful features.

**Installing Beaverlib** <br>
To install Beaverlib, you simply need to run the following command, replacing "path-from-root" 
with the path to the directory that you want to have the submodule saved in.
- `git submodule add git@https://github.com/beavertronics/beaverlib.git path-from-root`

**Updating Beaverlib** <br>
To update Beaverlib, you can update it in the terminal with the following command.

- `git submodule update --init --recursive --remote`

If using IntelliJ IDEA, you can update Beaverlib by:
[- Make sure IntelliJ IDEA is linked to Github.](https://www.jetbrains.com/help/idea/github.html)
- Open your project with the Beaverlib submodule.
- Click on "Branches" in the top-left.
- Select Beaverlib.
- Select the branch of Beaverlib you're using.
- Click "Update".

**Q&A** <br>
"Why isn't Beaverlib updating / showing up?"
- To fix this, you can delete the Beaverlib directory and then reinstall. Refer to the guide above, "Installing Beaverlib".
